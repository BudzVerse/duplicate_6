Dealing with Break Gauge

When preparing for battle, the first thing you should do is check the enemies you'll be facing and choose your skills and characters accordingly. For example, some bosses have shield regeneration abilities, so breaking their shield once isn't enough. You'll need to use shield-breaking skills like Crushing Blow repeatedly; otherwise, your attacks will deal reduced damage.

So, you need to keep using your shield-breaking ability during combat to keep the shield down. Repeatedly doing this can be tiring, but playing the game on EasyFun allows you to use the combo key feature with the custom key-mapping tool. To set it up:

Enter key-editing mode and right-click on the screen.

Drag one click-action button onto Crushing Blow and another onto a normal attack.

Assign both actions to a single keyboard key (e.g., the Spacebar).

Once you save the layout, you can spam a single key to activate both skills with one button spamming. In our scenario, as soon as Crushing Blow comes off cooldown, it will automatically activate while you're using normal attacks, ensuring the boss's shield doesn't recover.

Another great feature is that you can adjust your key-mapping setup in real-time. While staying in key-editing mode, you can change key assignments depending on your character's skills. If you use a fixed layout but don't remember the assigned keys, the key-toggle button will display the key layout on-screen, making it easy to play without guessing which key activates each skill.
